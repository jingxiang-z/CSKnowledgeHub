# File System Examples

Essential examples demonstrating file I/O and directory operations.

## C Examples

### ⭐⭐⭐ File Operations
- `file_operations.c` - **ESSENTIAL** - open(), read(), write(), close(), file descriptors

### ⭐⭐ Directory Operations
- `directory_traversal.c` - **Important** - opendir(), readdir(), closedir()

## Compilation & Running

### C Examples
```bash
gcc -o file_operations file_operations.c
./file_operations
```

### Python Examples
```bash
python3 file_operations.py
```

## Key Concepts Covered

- ✅ File I/O system calls (open, read, write, close)
- ✅ File descriptors and file table
- ✅ Directory operations (opendir, readdir, closedir)
- ✅ Directory traversal
- ✅ Understanding file system hierarchy
